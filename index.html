1. Limits 6 parts
① \(\displaystyle \lim_{x \to 0} \frac{\sin x}{x}\)
Solution: Use the standard trigonometric limit: \(\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1\). No further manipulation is needed. Answer: \(1\)
② \(\displaystyle \lim_{x \to 0} \frac{2x}{\tan x}\)
Solution: Rewrite as \(\displaystyle 2 \cdot \lim_{x \to 0} \frac{x}{\tan x}\). Since \(\tan x = \frac{\sin x}{\cos x}\), we have \(\displaystyle \frac{x}{\tan x} = \frac{x \cos x}{\sin x} = \frac{\cos x}{\sin x / x}\). Thus \(\displaystyle \lim_{x \to 0} \frac{x}{\tan x} = \frac{\cos 0}{1} = 1\). So the limit is \(2 \cdot 1 = 2\). Answer: \(2\)
③ \(\displaystyle \lim_{x \to 0} \frac{\sin 5x}{\sin 4x}\)
Solution: Multiply numerator and denominator by \(5x\) and \(4x\) respectively: \(\displaystyle \frac{\sin 5x}{\sin 4x} = \frac{5x}{4x} \cdot \frac{\sin 5x}{5x} \cdot \frac{4x}{\sin 4x}\). As \(x \to 0\), \(\frac{\sin 5x}{5x} \to 1\) and \(\frac{\sin 4x}{4x} \to 1\). Therefore the limit is \(\displaystyle \frac{5}{4} \cdot 1 \cdot 1 = \frac{5}{4}\). Answer: \(\frac{5}{4}\)
④ \(\displaystyle \lim_{x \to 0} \sin(x - \sin x)\)
Solution: As \(x \to 0\), \(\sin x \to 0\), so the inner expression \(x - \sin x \to 0\). Since \(\sin\) is continuous, \(\sin( \text{small} ) \to \sin 0 = 0\). Answer: \(0\)
⑤ \(\displaystyle \lim_{x \to 0} \frac{3x - \sin x}{x}\)
Solution: Split the fraction: \(\displaystyle \frac{3x}{x} - \frac{\sin x}{x} = 3 - \frac{\sin x}{x}\). Using the standard limit \(\frac{\sin x}{x} \to 1\), we get \(3 - 1 = 2\). Answer: \(2\)
⑥ \(\displaystyle \lim_{x \to 0} \frac{x - \sin x}{x^3}\)
Solution: Use the Taylor series expansion: \(\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\). Then \(x - \sin x = x - \left(x - \frac{x^3}{6} + \cdots\right) = \frac{x^3}{6} - \frac{x^5}{120} + \cdots\). Divide by \(x^3\): \(\displaystyle \frac{x - \sin x}{x^3} = \frac{1}{6} - \frac{x^2}{120} + \cdots\). As \(x \to 0\), all higher terms vanish, leaving \(\frac{1}{6}\). Answer: \(\frac{1}{6}\)
2. Continuity with parameter \(a\) 2 parts
① For what \(a\) is \(f(x) = \sqrt{a^2 - 2a}\) continuous at every \(x\)?
Solution: The function is constant (does not depend on \(x\)). A constant is continuous for all \(x\) as long as it is real‑valued. We need the radicand \(\ge 0\): \(a^2 - 2a \ge 0 \;\Rightarrow\; a(a-2) \ge 0\). This inequality holds when \(a \le 0\) or \(a \ge 2\). Answer: \((-\infty,\,0] \cup [2,\,\infty)\)
② For what \(a\) is \(g(x) = \dfrac{x}{a^2 - 2a}\) continuous?
Solution: This is a linear function in \(x\) divided by a constant denominator. It is continuous everywhere the denominator is non‑zero. Set \(a^2 - 2a \ne 0 \;\Rightarrow\; a(a-2) \ne 0 \;\Rightarrow\; a \ne 0\) and \(a \ne 2\). Answer: \(\mathbb{R} \setminus \{0, 2\}\)
3. Horizontal Asymptotes 2 parts
① \(\displaystyle f(x) = \frac{5x + 8x - 3}{3x^2 + 2}\)
Solution: Simplify numerator: \(5x+8x = 13x\), so \(f(x) = \frac{13x - 3}{3x^2 + 2}\). For horizontal asymptote, examine \(\lim_{x \to \infty} f(x)\) and \(\lim_{x \to -\infty} f(x)\). Divide numerator and denominator by the highest power \(x^2\): \(\displaystyle \frac{13/x - 3/x^2}{3 + 2/x^2}\). As \(x \to \infty\), \(13/x \to 0\), \(3/x^2 \to 0\), \(2/x^2 \to 0\), leaving \(0/3 = 0\). Answer: \(y = 0\)
② \(\displaystyle f(x) = \frac{11x + 2}{2x^2 - 1}\)
Solution: Degree of numerator = 1, degree of denominator = 2. Since degree(num) < degree(den), the horizontal asymptote is always \(y=0\). To verify: divide by \(x^2\): \(\displaystyle \frac{11/x + 2/x^2}{2 - 1/x^2} \to \frac{0}{2} = 0\). Answer: \(y = 0\)
4. Derivatives 9 parts
① \(y = 4 - x^2\)
Solution: Differentiate term‑by‑term: \(\frac{d}{dx}(4) = 0\), \(\frac{d}{dx}(-x^2) = -2x\). Answer: \(y' = -2x\)
② \(y = (2x+3)(5x^2 - 4x)\)
Solution: Use the product rule: \((uv)' = u'v + uv'\). Let \(u=2x+3,\; v=5x^2-4x\). \(u' = 2\), \(v' = 10x - 4\). Then \(y' = 2(5x^2-4x) + (2x+3)(10x-4)\). Expand: \(10x^2 - 8x + (20x^2 - 8x + 30x - 12) = 30x^2 + 14x - 12\). Answer: \(y' = 30x^2 + 14x - 12\)
③ \(y = x^2\)
Solution: Power rule: \(\frac{d}{dx} x^n = n x^{n-1}\). Here \(n=2\). Answer: \(y' = 2x\)
④ \(g(t) = \dfrac{1}{t^2}\)
Solution: Rewrite as \(g(t) = t^{-2}\). Power rule: \(g'(t) = -2 t^{-3} = -\dfrac{2}{t^3}\). Answer: \(g'(t) = -\dfrac{2}{t^3}\)
⑤ \(y = \dfrac{\sin x}{x}\)
Solution: Quotient rule: \(\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}\), with \(u = \sin x,\; v = x\). \(u' = \cos x\), \(v' = 1\). Thus \(y' = \frac{x \cos x - \sin x \cdot 1}{x^2}\). Answer: \(y' = \dfrac{x\cos x - \sin x}{x^2}\)
⑥ \(x^2 + y^2 = 25\) (implicit)
Solution: Differentiate both sides with respect to \(x\). Remember \(y\) is a function of \(x\). \(\frac{d}{dx}(x^2) = 2x\), \(\frac{d}{dx}(y^2) = 2y y'\), \(\frac{d}{dx}(25) = 0\). So \(2x + 2y y' = 0 \;\Rightarrow\; 2y y' = -2x \;\Rightarrow\; y' = -\frac{x}{y}\). Answer: \(y' = -\dfrac{x}{y}\)
⑦ \(f(x) = \dfrac{x}{2x+1}\)
Solution: Quotient rule: \(u = x,\; v = 2x+1\). \(u' = 1,\; v' = 2\). \(f'(x) = \frac{1\cdot(2x+1) - x\cdot 2}{(2x+1)^2} = \frac{2x+1-2x}{(2x+1)^2} = \frac{1}{(2x+1)^2}\). Answer: \(f'(x) = \dfrac{1}{(2x+1)^2}\)
⑧ \(y = (5x^2 - x^4)^7\)
Solution: Chain rule: \(\frac{d}{dx} [g(x)]^n = n [g(x)]^{n-1} \cdot g'(x)\). Let \(g(x) = 5x^2 - x^4\), \(n=7\). Then \(g'(x) = 10x - 4x^3\). \(y' = 7(5x^2 - x^4)^6 \cdot (10x - 4x^3)\). Answer: \(y' = 7(5x^2 - x^4)^6 (10x - 4x^3)\)
⑨ \(y = x^2 + \sin x\)
Solution: Sum rule: \(\frac{d}{dx}(x^2) = 2x\), \(\frac{d}{dx}(\sin x) = \cos x\). Answer: \(y' = 2x + \cos x\)
5. Tangent Slope & Increasing/Decreasing 2 parts
① Slope of tangent to \(y = x + \dfrac{2}{x}\) at \(x=-3\)
Solution: Find the derivative: \(y' = 1 - \dfrac{2}{x^2}\) (since \(\frac{d}{dx}(2x^{-1}) = -2x^{-2}\)). Evaluate at \(x = -3\): \(y'(-3) = 1 - \dfrac{2}{(-3)^2} = 1 - \dfrac{2}{9} = \dfrac{7}{9}\). The slope of the tangent line is the derivative at that point. Answer: Slope \(= \dfrac{7}{9}\)
② Critical points & intervals of \(f(x) = x^3 - 12x - 5\)
Solution: Find \(f'(x) = 3x^2 - 12\). Set \(f'(x) = 0\): \(3x^2 - 12 = 0 \;\Rightarrow\; x^2 = 4 \;\Rightarrow\; x = \pm 2\). These are the critical points. Test intervals:
For \(x < -2\) (e.g., \(x=-3\)): \(f'(-3) = 27-12 = 15 > 0\) → increasing.
For \(-2 < x < 2\) (e.g., \(x=0\)): \(f'(0) = -12 < 0\) → decreasing.
For \(x > 2\) (e.g., \(x=3\)): \(f'(3) = 27-12 = 15 > 0\) → increasing. Critical points: \(x = -2,\; 2\). Increasing on \((-\infty, -2) \cup (2, \infty)\); Decreasing on \((-2, 2)\).
7. Integrals 4 parts
① \(\displaystyle \int \frac{4}{\sin x} \, dx\)
Solution: Recall that \(\frac{1}{\sin x} = \csc x\). So the integral is \(4 \int \csc x \, dx\). Standard formula: \(\int \csc x \, dx = \ln\left|\tan\frac{x}{2}\right| + C\). Multiply by 4. Answer: \(4 \ln\left|\tan\frac{x}{2}\right| + C\)
② \(\displaystyle \int \frac{5}{\sin^2 x} \, dx\)
Solution: \(\frac{1}{\sin^2 x} = \csc^2 x\). So the integral is \(5 \int \csc^2 x \, dx\). Standard formula: \(\int \csc^2 x \, dx = -\cot x + C\). Multiply by 5. Answer: \(-5\cot x + C\)
③ \(\displaystyle \int (x^3 + 1)^5 (3x^2 + 1) \, dx\)
Solution: Note: The derivative of \(x^3 + x\) is \(3x^2 + 1\). This suggests the base should be \(x^3 + x\) (likely a typo). Assuming the integrand is \((x^3 + x)^5 (3x^2 + 1)\), let \(u = x^3 + x\), then \(du = (3x^2+1)dx\). The integral becomes \(\int u^5 \, du = \frac{u^6}{6} + C\). Substitute back: \(\frac{(x^3 + x)^6}{6} + C\). If the literal expression \((x^3+1)^5(3x^2+1)\) is required, expand \((x^3+1)^5\) and multiply by \((3x^2+1)\), then integrate term‑by‑term (very lengthy). Answer (with correction): \(\dfrac{(x^3 + x)^6}{6} + C\)
④ \(\displaystyle \int \frac{3}{\sqrt{x^2 + 1}} \, dx\)
Solution: Factor out 3: \(3 \int \frac{1}{\sqrt{x^2 + 1}} \, dx\). Standard formula: \(\int \frac{dx}{\sqrt{x^2 + 1}} = \sinh^{-1}(x) + C = \ln\left|x + \sqrt{x^2+1}\right| + C\). Multiply by 3. Answer: \(3\sinh^{-1}(x) + C = 3\ln\left|x + \sqrt{x^2+1}\right| + C\)
9. More Integrals 3 parts
① \(\displaystyle \int \sqrt{x + 1} \, dx\)
Solution: Let \(u = x+1\), then \(du = dx\). The integral becomes \(\int u^{1/2} \, du = \frac{u^{3/2}}{3/2} + C = \frac{2}{3} u^{3/2} + C\). Substitute back \(u = x+1\). Answer: \(\dfrac{2}{3}(x+1)^{3/2} + C\)
② \(\displaystyle \int \sec^2(x+1) \cdot \sec x \, dx\)
Solution: This integral has no elementary antiderivative in its given form because it is a product of secant functions with different arguments. Correction: If the second factor was \(\tan(x+1)\), i.e., \(\int \sec^2(x+1) \tan(x+1) dx\), then let \(u = \tan(x+1)\), \(du = \sec^2(x+1) dx\), giving \(\int u \, du = \frac{u^2}{2} + C = \frac{1}{2}\tan^2(x+1) + C\). Please verify the original problem statement. Answer: No elementary antiderivative (likely a typo). Corrected version gives \(\frac{1}{2}\tan^2(x+1) + C\).
③ \(\displaystyle \int x^2 e^{x^3} \, dx\)
Solution: Let \(u = x^3\), then \(du = 3x^2 dx \;\Rightarrow\; x^2 dx = \frac{du}{3}\). The integral becomes \(\int e^u \cdot \frac{du}{3} = \frac{1}{3} e^u + C\). Substitute back \(u = x^3\). Answer: \(\dfrac{1}{3} e^{x^3} + C\)
